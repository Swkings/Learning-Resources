1.  translate 与replace

SELECT translate('我是中国人,我爱中国', '中国', 'China') ,

               translate ('I am Chinese, I love China', 'China', '中国') "Translate example"

FROM DUAL;

 

示例证明是按照字符来处理，不是按照字节来处理，如果to_string的字符数比from_string多的话，多出的字符数似乎没有什么用处，也不会引发异常。

 

将translate换成replace函数，比较结果：

SELECT replace('我是中国人,我爱中国', '中国', 'China') ,

               replace('I am Chinese, I love China', 'China', '中国') "Replace example"

FROM DUAL;

 

2. 在银行转帐时经常看见账户人只显示姓名的最后一个字，其余的用星号代替，可用replace函数实现：

SELECT name, replace(name,substr(name,1,length(name) - 1),rpad('*',length(name)-1,'*')) xm1

FROM s;

 

3.  将所有姓“刘”、“陈”和“罗”的学生的注册日期提前一年。

update reg

set regdate = add_months(regdate,12)

where sno in (select sno from s where substr(name,1,1) in ('刘','陈','罗'));

 

4. 将所有“网络工程”的女学生的注册日期改成当月的1号。

update reg

set regdate = trunc(regdate,'month')

where sno in (select sno from s where speciality='网络工程' and gender='女');

 

5.  按年、按月、按周、按日进行统计分析

.按年

select to_char(regdate,'yyyy') 年份 ,count(*) 注册人数

from reg

group by to_char(regdate,'yyyy')

order by 年份;

 

.按月

select to_char(regdate,'yyyy-mm') 月份 ,count(*) 注册人数

from reg

group by to_char(regdate,'yyyy-mm')

order by 月份;

 

.按周

select to_char(regdate,'yyyy') 年份,to_char(regdate,'iw') 周 ,count(*) 注册人数

from reg

group by to_char(regdate,'yyyy'),to_char(regdate,'iw')

order by 年份,周;

 

6. 在SQL疑难问题中，DECODE函数常常发挥非常灵活的作用。其中的一个技术就是为了某种目的可以将一个表的行转换成列。例如：

SELECT to_char(regdate,'YYYY') year,   count(*) numofstudent

FROM reg 

WHERE to_char(regdate,'YYYY') in ('2013','2014','2015');

 

显示结果：

YEAR NUMOFSTUDENT          

---- ----------------------

2014  17                    

2015  10                    

2013  18                    

 

利用DECODE，改写为：

SELECT sum(decode(to_char(regdate,'YYYY'),'2013',1,0)) "2013",

               sum(decode(to_char(regdate,'YYYY'),'2014',1,0)) "2014",

               sum(decode(to_char(regdate,'YYYY'),'2015',1,0)) "2015"

from reg

where to_char(regdate,'YYYY') in ('2013','2014','2015');

 

显示结果：

2013                   2014                   2015                  

-------------------- ----------------- ----------------------

18                      17                      10
